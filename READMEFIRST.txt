A continuación les presentamos un pequeño manual de como utilizar
la aplicación. 

Si se tiene instalado Java en el ordenador, tiene dos opciones:
a. Por un lado puede ejecutarlo desde un desarrollador de Java como puede
ser NetBeans o Eclipse Gavab. Bastará con importar el proyecto 
a su entorno de trabajo, compilarlo y ya estará listo para usarse.

b. Por otro lado puede directamente abrir el proyecto desde el .JAR
que se encuentra en Buscaminas/dist. Esto permitirá que la aplicación
se ejecute directamente y pueda jugar sin necesidad de un entorno.

Si por el contrario, no tiene instalado Java en su ordenador, en esta
misma carpeta encontrará el ejecutable que le permitirá instalarlo y
disfrutar de nuestro juego.

Así mismo en la carpeta Buscaminas/dist/javadoc tiene toda la documentación
del software.

Así mismo podrá encontra los proyectos de software y de gestión
en esta misma carpeta.
